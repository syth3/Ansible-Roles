Role Name
=========

Install Jetbrains Toolbox on a linux system. After this playbook has completed, run /opt/jetbrains-toolkit/jetbrains-toolkit to start the toolkit.

Requirements
------------

None.

Role Variables
--------------

None.

Dependencies
------------

None.

Example Playbook
----------------

```
- hosts: linux-workstations
  roles:
      - install-jetbrains-toolbox
```
